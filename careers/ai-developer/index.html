AI/ML Developer

📍 Remote 💼 Full-time or Contract 🌍 Worldwide

About This Role

Join our team to build AI-powered products that help healthcare and legal professionals work smarter. You'll architect RAG pipelines, integrate LLMs, and create APIs that power our assistant network.

This is a high-impact engineering role for developers passionate about AI. Work full-time or as a contract specialist with competitive compensation.

Responsibilities

  • • Design and build RAG (Retrieval-Augmented Generation) pipelines
  • • Integrate LLMs (GPT, Claude, open-source models) into applications
  • • Develop REST APIs and webhook services
  • • Build data pipelines and ETL processes
  • • Optimize models for latency and cost
  • • Write clean, testable, production-ready code
  • • Collaborate with product and operations teams

Requirements

Required

  • • Strong JavaScript/Node.js OR Python skills (or both)
  • • Experience with LLM APIs (OpenAI, Anthropic, etc.)
  • • Understanding of RAG concepts and vector databases
  • • RESTful API design and development experience
  • • Experience with async programming and background jobs
  • • Comfortable with Git, testing, and CI/CD
  • • Problem-solving mindset and attention to detail

Nice to Have

  • • Experience with vector DBs (Pinecone, Weaviate, Milvus)
  • • Knowledge of healthcare or legal domain
  • • Experience with LangChain or LlamaIndex
  • • Machine learning background
  • • Docker and cloud deployment experience (AWS, GCP)

Benefits

  • ✓ Work on cutting-edge AI and LLM projects
  • ✓ Full remote work with flexible schedule
  • ✓ Full-time or contract opportunities
  • ✓ Competitive compensation
  • ✓ Build products impacting real professionals
  • ✓ Access to latest AI models and tools
